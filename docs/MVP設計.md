# MVP 設計 — ポケカ買取価格サーチ

## 目的

ユーザーがポケモンカードの**買取価格**を、複数ショップで素早く比較できること。

MVP では全カタログではなく、**最新1〜2セット + 人気カードのサンプル**で動作確認する。

---

- カード名（部分一致）または番号（001, 025 等）で検索

- 結果テーブル: サムネ / 名前 / セット / 番号 / 最高買取 / 更新日
- 行クリック → カード詳細へ

## DB 設計（SQLite）

### ER 図（概念）

```
sets ──< cards ──< buyback_prices >── shops
```

### テーブル定義

#### `sets` — 拡張パック

| カラム | 型 | 説明 |
|--------|-----|------|
| id | TEXT PK | 内部ID（例: sv9） |
| tcgdex_id | TEXT | TCGdex ID（例: SV9） |
| name | TEXT | セット名（日本語） |
| series_name | TEXT | シリーズ名 |
| release_date | TEXT | 発売日 ISO |
| card_count_official | INTEGER | 公式カード数 |

#### `cards` — カードマスタ

| カラム | 型 | 説明 |
|--------|-----|------|
| id | TEXT PK | 内部ID（例: sv9_033） |
| set_id | TEXT FK | sets.id |
| tcgdex_id | TEXT | TCGdex ID（例: SV9-033） |
| local_id | TEXT | カード番号（003, 033） |
| name | TEXT | カード名 |
| rarity | TEXT | レアリティ |
| image_url | TEXT | 画像ベースURL |
| search_text | TEXT | 検索用テキスト |

#### `shops` — 買取ショップ

| カラム | 型 | 説明 |
|--------|-----|------|
| id | TEXT PK | cardrush / mercard / fullcomp / magi |
| name | TEXT | 表示名 |
| slug | TEXT | URL用 |
| website_url | TEXT | トップページ |

#### `buyback_prices` — 買取価格

| カラム | 型 | 説明 |
|--------|-----|------|
| id | INTEGER PK | 自動採番 |
| card_id | TEXT FK | cards.id |
| shop_id | TEXT FK | shops.id |
| price_yen | INTEGER | 買取価格（円） |
| condition_note | TEXT | 状態（美品 等） |
| as_of_date | TEXT | **価格更新日**（ISO date） |
| product_url | TEXT | ショップ商品ページ |

**ユニーク制約:** `(card_id, shop_id, as_of_date)` — 同一日の重複防止

#### `price_history` — 価格履歴（グラフ用）

| カラム | 型 | 説明 |
|--------|-----|------|
| id | INTEGER PK | 自動採番 |
| card_id | TEXT FK | cards.id |
| shop_id | TEXT FK | shops.id |
| price_yen | INTEGER | 価格（円） |
| recorded_at | TEXT | 取得日時（ISO） |
| condition_note | TEXT | 状態・参考ラベル |
| product_url | TEXT | 商品/出品ページ |

取得のたび追記。`buyback_prices` は最新スナップショット、`price_history` は推移用。

## 技術スタック

| 層 | 採用 |
|----|------|
| Web | FastAPI + Uvicorn |
| テンプレート | Jinja2 |
| DB | SQLite |
| カードマスタ | TCGdex API（任意連携） |
| 買取価格 | サンプル seed / Magi は `fetch_magi_prices.py` |

**比較対象ショップ（MVP）:** カードラッシュ / メルカード / フルコンプ / **Magi**（4店舗）。Magi は C2C 最安出品を参考価格として表示。リンクは `https://magi.camp/items/search?forms_search_items[keyword]=...` 形式。

（11_名称未定Web作り と同じ構成）
