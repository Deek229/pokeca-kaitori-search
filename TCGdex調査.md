# TCGdex API 調査メモ

調査日: 2026-06-20  
API ベースURL: `https://api.tcgdex.net/v2/{lang}/`

## 概要

TCGdex はポケモンカードのオープンAPI。認証不要で JSON を返す。  
日本語ロケール `ja` が用意されており、**カード名・セット名・説明文が日本語**で取得できる。

---

## エンドポイント一覧（主要）

| 用途 | URL 例 | 備考 |
|------|--------|------|
| セット一覧 | `GET /v2/ja/sets` | 全セットの id, name, cardCount |
| セット詳細 | `GET /v2/ja/sets/SV9` | セット内カード一覧付き |
| カード詳細 | `GET /v2/ja/cards/SV9-033` | 攻撃・HP・レアリティ等 |
| 名前検索 | `GET /v2/ja/cards?name=ピカチュウ` | 部分一致っぽいマッチ |
| 画像 | レスポンスの `image` フィールド | `/low.webp` `/high.webp` を末尾に付与 |

---

## 取得できるカードメタデータ

### セット一覧 (`/sets`)

```json
{
  "id": "SV9",
  "name": "バトルパートナーズ",
  "cardCount": { "total": 132, "official": 100 }
}
```

### セット詳細 (`/sets/{id}`)

- `id`, `name`, `releaseDate`, `serie`（シリーズ名）
- `cards[]`: 各カードの `id`, `localId`, `name`, `image`（任意）
- `cardCount`: official / total / holo 等
- `legal`: standard / expanded フラグ

### カード詳細 (`/cards/{id}`)

実際に取得した `SV9-001`（キャタピー）の例:

| フィールド | 内容 |
|-----------|------|
| `name` | キャタピー（日本語） |
| `localId` | 001 |
| `set` | バトルパートナーズ |
| `rarity` | Common |
| `category` | Pokemon |
| `image` | `https://assets.tcgdex.net/ja/SV/SV9/001` |
| `hp`, `types`, `attacks`, `weaknesses` | ゲーム情報あり |
| `description` | 日本語フレーバーテキスト |
| `dexId` | 図鑑番号 |
| `regulationMark` | レギュレーション（I 等） |
| `illustrator` | イラストレーター名 |
| `updated` | 最終更新日時 |
| `pricing` | Cardmarket（EUR）のみ。日本向け買取価格は**なし** |

### 画像URL

- ベース: `https://assets.tcgdex.net/ja/SV/SV9/033`
- サムネ: `{base}/low.webp`
- 高解像度: `{base}/high.webp`

## 日本語サポート

| 項目 | 対応状況 |
|------|---------|
| セット名 | ✅ 日本語（例: バトルパートナーズ、ポケモンカード151） |
| カード名 | ✅ 日本語（例: リーリエのピッピex、Nのゾロアークex） |
| 説明文 | ✅ 日本語フレーバーテキストあり |
| 攻撃名 | ✅ 日本語（例: むしくい） |
| シリーズ名 | ✅ 日本語（スカーレット&バイオレット 等） |
| レアリティ | △ 英語表記（Common, Double Rare, Art Rare 等） |
| 買取/相場価格 | ❌ 海外 Cardmarket のみ（EUR）。日本の買取価格は別途取得が必要 |

**結論:** カードマスタ（名前・番号・画像・セット）の日本語利用には **十分使える**。  
買取価格比較サイトの「価格データ」は TCGdex では賄えない。

## 本プロジェクトでの使い方

1. **カードマスタ取得:** `tools/fetch_tcgdex_set.py SV9` で1セット分をDBに投入可能
2. **検索補助:** 名前検索APIで候補カードを探せる
3. **画像表示:** `image` + `/low.webp` でサムネ表示
4. **買取価格:** 別途ショップスクレイピング or 手動入力（seed.py のサンプル）

---

## 制限・注意点

- レート制限の明示ドキュメントは要確認。連続大量リクエストは避ける
- 一部セットIDに重複エントリが見られる（API側データ品質）
- 古いセットは画像URLがないカードあり
- `pricing.tcgplayer` は日本版カードでは null になりがち
- 買取価格サイトには **TCGdex = カード辞書**、**価格 = 自前収集** の役割分担が適切
